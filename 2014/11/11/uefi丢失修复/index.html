uefi丢失修复

发表于 2014-11-11 | 更新于 2019-02-14

1、我们假设系统根目录是在/dev/sda4

2、挂载系统目录

mount /dev/sda4 /mnt

3、挂载其他目录：

for f in proc sys dev ; do mount –bind /$f /mnt/$f ; done

3、查询历史的efi，并删除不需要的启动项，我们假设假设需要删除Boot0000*

efibootmgr -v
efibootmgr -b 0000* -B

4、修复bootloader

chroot /mnt
grub-install –bootloader-id ubuntu /dev/sda
